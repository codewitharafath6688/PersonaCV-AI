### 🤖 PersonaCV AI (Backend)

PersonaCV AI is a backend service designed to generate, improve, and manage professional resume content using AI. It also provides career guidance and intelligent responses to user queries related to jobs, CV building, and career growth.

This backend acts as the core intelligence layer for the PersonaCV AI system, powering chatbot interactions and resume optimization features.

* **Live Server:** https://persona-cv-ai-agent-backend.vercel.app/

* 🎯 Features
* 🧠 AI-powered resume generation and improvement
* 💬 Career guidance chatbot responses
* 📄 Structured CV content handling
* ⚡ Fast API responses via OpenRouter AI models
* 🔐 Secure backend architecture
* 🔗 Easily connectable to any frontend (React / Next.js)

### 🛠️ Tech Stack
Node.js – Backend runtime
Express.js – API framework
OpenRouter AI API – LLM integration (GPT / Claude / etc.)
MongoDB – Database for storing users & resumes
dotenv – Environment configuration
CORS – Frontend-backend communication
Axios / Fetch – API requests handling

### 📌 Role of Backend

This project is strictly a backend service, responsible for:

Processing user prompts
Communicating with OpenRouter AI models
Returning structured AI responses
Managing resume data and chat history

### 🚀 Example Use Cases
“Improve my resume for software engineer role”
“Write a professional summary for a fresher”
“What skills should I add for React developer job?”
